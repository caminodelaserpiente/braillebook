# BRAILLEBOOK
Braille PDF Generation. 

## Installation
```consol
pip install -r requirements.txt
```


## Run interface
```consol
streamlit run interface.py
```
    You can now view your Streamlit app in your browser.    
    
    URL: http://localhost:8501
